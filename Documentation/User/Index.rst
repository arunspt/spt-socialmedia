Users Manual
============

This is a very simple and easy configurable social media widget extension. User can manage the predefined social media items in the backend.

1. Install the extension using extension manager.

2. Include the extension in root page static template.

3. Edit root page properties.

4. In the extended social media tab, user can add the link title, icon and link.

5. Rather than social media items, user can add internal link, external link, file, phone etc.
